# next-store

A proof of concept using a NextJS seed ready to be deployed with Vercel.

### Features

- Lazy Image Loading
- CSS Purged

### Tech Stack

- Nextjs
- PurgeCSS
- TailwindCSS
- PostCSS
  - Allows nesting and imports.

## Up and Running

- Develop running `npm run dev` - [http://localhost:3000](http://localhost:3000)

- Build `npm run build`

- Deploy `vercel --prod`
